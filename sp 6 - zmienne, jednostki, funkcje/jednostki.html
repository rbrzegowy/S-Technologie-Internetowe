Jednostki bezwzględne

  • Najczęściej używana: px (piksele CSS)
  • Do druku przydają się też pt, cm, mm, in
  • Bezwzględne wartości są przewidywalne, ale mniej elastyczne
width: 220px;
font-size: 18pt;
margin-top: 1cm;

2

Wielkość tekstu

  • rem to mnożnik rozmiaru fontu na elemencie html
  • em to mnożnik rozmiaru fontu rodzica
  • % przy font-size to procent wartości rodzica
html { font-size: 16px; }
.lead { font-size: 1.25rem; }

To jest 1.25rem

Rodzic 20px

To jest 1.5em (30px)

To jest 125%

3

Jednostki względne i viewport

  • % odnosi się do wymiaru rodzica
  • ch bazuje na szerokości znaku "0" - fajne do definiowania/ograniczania szerokości kolumny tekstu
  • vw i vh liczą się od szerokości i wysokości okna przeglądarki
  • vmin to mniejszy wymiar okna, a vmax większy
.card { width: 50%; }
.copy { max-width: 60ch; }
.hero { width: 18vw; height: 32vh; border-radius: 6vmin; }

60ch

18vw

Kolory i przezroczystość

  • Najczęściej - hex lub rgb
  • hsl wygodnie zmienia odcień i nasycenie
  • oklch daje bardziej przewidywalną percepcję koloru - zobacz zmianę jasności w porównaniu z hsl
  • Przezroczystość - kanał alpha lub właściwością opacity
  • Świetne wyjaśnienie różnic: evilmartians.com/chronicles/oklch-in-css-why-quit-rgb-hsl
