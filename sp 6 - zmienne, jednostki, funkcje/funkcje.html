Funkcje w CSS

1

Kolory i gradienty

  • color-mix() miesza dwa kolory w zadanej proporcji
  • linear-gradient() i conic-gradient() tworzą tła i akcenty
  • from() z calc() pozwala tworzyć nowe odcienie na bazie koloru źródłowego
background: color-mix(in srgb, oklch(62% 0.18 260) 65%, white);
background: linear-gradient(135deg, #f56b45, #2d7ff9);
background: linear-gradient(135deg, oklch(from #4f7cff calc(l + 0.1) c h), oklch(from #4f7cff calc(l - 0.08) c h));

linear-gradient()

2

Funkcje stosowane w filter()

  • blur() rozmywa element
  • brightness(), contrast(), saturate() - zmiana jasności, kontrastu i nasycenia
  • hue-rotate() i drop-shadow() - zmiana odcienia i dodanie cienia
filter: blur(2px) saturate(1.2);
filter: brightness(1.2) contrast(1.25);
filter: hue-rotate(35deg) drop-shadow(0 8px 18px rgba(46, 91, 255, 0.35));
blur() + saturate()
brightness() + contrast()
hue-rotate() + drop-shadow()
3

Rozmiary i ograniczenia

  • min() bierze mniejszą z podanych wartości
  • max() pilnuje minimalnego progu rozmiaru
  • clamp() trzyma wartość pomiędzy min i max
  • minmax() definiuje zakres rozmiaru (np. dla kolumn i kart)
width: min(68ch, 100%);
font-size: clamp(1rem, 0.85rem + 1.2vw, 1.8rem);
padding: max(0.75rem, 2vw);
grid-template-columns: minmax(180px, 320px);

4

Praktyczne użycie url()

  • url() wczytuje grafikę jako tło, maskę lub ikonę
  • url() w @font-face ładuje plik fontu
  • Ładowanie zasobów zewnętrznych można też zrealizować w html za pomocą link:
    <link href="https://fonts.googleapis.com/css2?family=Cormorant" rel="stylesheet">
background-image: url('./assets/noise.png');
@import url('https://fonts.googleapis.com/css2?family=Cormorant+Garamond:wght@500;700&display=swap');
.font-tile { font-family: 'Cormorant Garamond', Georgia, serif; }
url() dla grafiki tła
url() dla fontu (@font-face)
5

attr() - style na bazie atrybutu

  • attr() może pobierać wartości z atrybutów elementu
  • Pełne wsparcie rynku dla attr() - właściwość content
  • Uwaga na wsparcie attr() w pozostałych właściwościach css (np. color: attr())
font-size: attr(data-size rem, 1rem);
color: attr(data-color color, #2a3f58);
transform: skewX(attr(data-rotation deg, 0deg));
Atrybut: spokojny
Atrybut: dynamiczny
Atrybut: akcent

Siatka i układ

  • repeat() upraszcza definicję kolumn i wierszy
  • minmax() daje elastyczne granice dla tracków
  • fit-content() ogranicza szerokość kolumny do sensownego maksimum
grid-template-columns: repeat(3, minmax(120px, 1fr)) fit-content(220px);
