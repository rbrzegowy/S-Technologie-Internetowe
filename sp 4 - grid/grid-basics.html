1

Włączenie grid

  • Kontener staje sie gridem
  • Brak definicji wierszy i kolumn
  • Domyślnie - układ jednokolumnowy
  • Nowe wiersze/kolumny tworzone są automatycznie (grid-auto-flow)
  • Pomiędzy pojemnikami można zdefiniować odstęp (gap)
display: grid;
gap: 12px;
grid-auto-flow: row;

Wyrównanie w gridzie

  • justify-items i align-items wyrównują zawartość wewnątrz pojedynczych komórek
  • justify-content i align-content ustawiają położenie całej siatki w kontenerze
  • Pojedynczy element może nadpisać wyrównanie przez justify-self i align-self
  • Skróty do justify- i align-: place-content, place-items, place-self
justify-items: end; align-items: center;
justify-content: space-evenly; align-content: space-evenly;

Pozycjonowanie elementów

  • Elementy można pozycjonować przez podanie numeru linii - start/koniec
  • Linie można też nazwać (i używać nazw zamiast numerów)
  • Dwa lub więcej elementów może zajmować ten sam obszar (i nachodzić na siebie)
  • Warstwy kontrolujemy przez z-index
grid-template-columns: [full-start] 1fr [content-start] 1fr [center-start] 1fr [content-end] 1fr [aside-start] 1fr [aside-end] 1fr [full-end];
.a { grid-column: full-start / content-end; grid-row: top / lower; }
.b { grid-column: center-start / aside-end; grid-row: middle / bottom; z-index: 2; }
.d { justify-self: end; align-self: start; }
A: full-start / content-end
B: center-start / aside-end
C: aside-start / full-end
D: self -> end / start

Dodatkowe funkcje do definiowania siatki

  • repeat() - powtarzanie kolumn lub wierszy
  • minmax() - minimalna/maksymalna wartość (tutaj - szerokość)
  • fit-content(max) - szerokość dopasowana do zawartości, nie większa niż max
grid-template-columns: repeat(2, minmax(140px, 1fr));
grid-template-columns: fit-content(220px);
grid-template-rows: minmax(70px, auto);
clamp(80px, 12vw, 120px) max(70px, 8vw);

Brak zawartości dla zdefiniowanych pojemników

  • Rozwiązanie - funkcje auto-fit i auto-fill
  • auto-fill utrzymuje puste kolumny, nawet gdy brak elementów
  • auto-fit zwija puste kolumny i rozszerza istniejące
grid-template-columns: repeat(auto-fill, minmax(120px, 1fr));
grid-template-columns: repeat(auto-fit, minmax(120px, 1fr));
